# Team_message
 discord bot

Linux installation:
```
git clone https://github.com/Emzime/Team_message.git
cd Team_message
python -m venv venv
. ./venv/bin/activate
pip install discord PyGithub
python main.py
```

Windows installation:
```
git clone https://github.com/Emzime/Team_message.git
cd Team_message
py -m venv venv
. ./venv/bin/activate
pip install discord PyGithub
py main.py
```
